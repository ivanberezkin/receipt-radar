**Project Status:⚠️ WORK IN PROGRESS**

### 🛠 Tech Stack

### Frontend <br>
* **Framework:** React 19 (Functional Components & Hooks)<br>
* **Build Tool:** Vite (Started as CRA, but migrated to Vite for optimized performance).<br>
* **Language:** TypeScript.<br>
* **State Management:** Custom Hooks architecture (useReceiptForm, useReceipts) for clean separation of concerns.<br>
* **Testing:** Vitest & React Testing Library with MSW (Mock Service Worker).<br>
* **Data fetching:** Axios.

### Backend<br>
* **Framework:** Java / Spring Boot<br>
* **Data Access:** Spring Data JPA (Hibernate)
* **Database:** PostgreSQL
* **Architecture:** RESTful API.<br>
* **Server:** Currently only running on localhost.<br>
